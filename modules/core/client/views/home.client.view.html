Om Stipendify

Detta system är ett open source projekt utvecklat som en övning för Shape App att lära sig utveckla meanjs-projekt. Målet med projektet är att bygga ett system för att underlätta stipendiumansökningar och översikt. Det finns ett inloggningssystem som tillåter att information sparas för de sökande nästa termin och underlättar deras ansökning. Projektet är gratis för alla att utnyttja och vidareutveckla på egen risk. ShapeApp ansvarar inte för er användning av systemet.

För Studentföreningar

Om din studentförening är intresserade av att utnyttja systemet så kör på. Be era IT-ansvariga att ladda ner koden från github och köra igång den på er server. ShapeApp kan erbjuda hosting för er, då tillkommer en avgift som får förhandlas. Kontakta i sådana fall: c@shapeapp.se. Vid andra frågor, buggar ellerönskan om funktioner kontakta återigen: c@shapeapp.se

För utvecklare

Stipendify är byggt med MEANjs och yeoman för att generera koden. Koden till detta projekt finns på github; https://github.com/cNille/stipendify. Jag antar att ni sen innan känner till npm och grunt för att kunna få igång projektet. Om inte, eller om ni har andar frågor får ni kontakta mig på c@shapeapp.se
