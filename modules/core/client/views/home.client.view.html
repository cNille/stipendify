Om Stipendify

Detta system är ett open source projekt utvecklat som en läxa för Shape App att lära sig meanjs-projekt. Koden till detta projekt finns på github; https://github.com/cNille/stipendify. Där finns det instruktioner om hur du laddar ner programmet och startar upp den på din dator eller din server. Vid problem och frågor kan ni kontakta: c@shapeapp.se
